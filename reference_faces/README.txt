REFERENCE FACES FOR PERSON RECOGNITION
============================================================

Add reference photos to these folders:

1. bride-groom/
   - Add clear photos of the bride and/or groom
   - Can be individual or together
   - Face should be clearly visible

2. bride-parents/
   - Add photos of bride's mother and/or father
   - Clear face shots preferred

3. bride-siblings/
   - Add photos of bride's brothers/sisters
   - One photo per sibling

4. groom-parents/
   - Add photos of groom's mother and/or father
   - Clear face shots preferred

TIPS:
- Use clear, well-lit photos
- Face should be clearly visible (no sunglasses, side angles)
- One face per photo is ideal
- JPG, JPEG, PNG formats supported
- You can add multiple reference photos per category
